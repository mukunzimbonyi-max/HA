# My Health My Friend (HA)

My Health My Friend is a digital platform designed to give Rwandan youth safe, private, and stigma-free access to sexual and reproductive health (SRH) information and services.

## Core Features
- **AI-Powered Assistant**: Ask SRH and mental health questions privately.
- **GPS-Integrated Directory**: Locate nearby health centers, pharmacies, and contraceptive services.
- **Professional Booking**: Connect directly with qualified reproductive health professionals and psychologists.
- **Multilingual Video Library**: Educational content in English and Kinyarwanda with accessibility features (subtitles, sign language).
- **Masterclass Cohorts**: Join biannual online masterclasses to learn and grow.

## Tech Stack
- **Frontend**: React, Vite, Framer Motion, Lucide React
- **Backend**: Node.js, Express, PostgreSQL
- **AI**: Google Gemini API

## Setup
1. Clone the repository
2. Install dependencies: `npm install` (root, frontend, and backend)
3. Configure `.env` files in both `frontend` and `backend`
4. Start development: `npm run dev`
